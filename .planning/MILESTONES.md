# Project Milestones: Piak Backend

## v1.0 MVP (Shipped: 2026-01-12)

**Delivered:** AI-powered contact organizer backend with natural language parsing and semantic search.

**Phases completed:** 1-5 (8 plans total)

**Key accomplishments:**

- Claude API integration with ClaudeService class, Haiku 4.5 model, and Firebase secret management
- Contact data model with dual raw+structured approach (rawNote always preserved)
- Intent classification routing user input to add/update/search operations
- Contact extraction parsing both voice-to-text and typed shorthand
- AI-powered semantic search ("coffee shop" matches "starbucks")
- Firestore security rules with owner-based access control
